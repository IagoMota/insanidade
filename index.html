Oque é insanidade?

Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa. Programação pode ser feita usando uma variedade de "idiomas" do computador, tais como SQL, Java, Python e C++. Versão original criada por Pamela Fox

Porque programação pode ser considerada insanidade?

Do nada varias pessoas se reunem para fazer um site, na area da programação você vai de uma beldade de capacidade cognitiva extremamente avançada a uma ameba em questões de segundos, por isso é insano.
